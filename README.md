# Emotion-Aware Conversational AI

This project builds a fine-tuned emotional support chatbot using open-source LLMs.

Datasets used:
- Emotion Dataset
- ESConv Dataset

Steps:
1. Data cleaning
2. Model fine-tuning
3. Safety layer implementation
4. Evaluation

Model: Mistral 7B

Safety layer detects:
- suicide ideation
- self-harm intent
- extreme hopelessness
